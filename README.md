# Admin Panel

Back-office admin panel with CRUD operations, sortable data tables, forms, and dashboard charts. Built with Next.js, shadcn/ui, TanStack Table, and Recharts.

**Stack:** Nextjs | **Complexity:** moderate
**Tags:** `admin`, `crud`, `data-table`, `forms`, `charts`, `dashboard`, `shadcn`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
